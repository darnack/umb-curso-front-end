Las tablas son un potente elemento que permiten organizar eficientemente un conjunto de datos en filas y columnas, permite la lectura y comparación rápida de los datos proporcionados.

Estructura de una tabla

La etiqueta principal <table> contiene las filas <tr> de la tabla, y cada fila contiene sus columnas; de cabecera <th>, y de campos <td>.

Expansión de celdas

Las celdas pueden expandirse para ocupar más de una fila o columna. Esto es útil para agrupar datos.

Para expandir una celda horizontalmente se debe utilizar el atributo colspan e indicar la cantidad de columnas a expandir.

Para expandir una celda verticalmente se debe utilizar el atributo rowspan e indicar la cantidad de filas a expandir.

Lo que daría como resultado...

columna 1 columna 2 columna 3
dato 1 dato 2
dato 3 dato 4

Atributos

Además existen algunos atributos que permiten determinar el aspecto de las tablas:

Atributo Descripción
align Determina la alineación horizontal de una tabla o celda.
valign Determina la alineación vertical de una tabla o celda.
cellpadding Determina el margen interior de las celdas.
cellspacing Determina el espacio o margen de las celdas.
width Determina el ancho de una tabla o celda.
height Determina la altura de una tabla o celda.
border Determina el grosor, color y tipo de los bordes de una tabla o celda individual.
bgcolor Determina el color de una celda.
