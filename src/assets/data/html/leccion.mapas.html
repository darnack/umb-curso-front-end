Los mapas de imágenes permiten crear áreas distinguibles dentro de una misma imagen, y a cada área se le puede asignar un enlace. Esto resulta muy útil para crear infografías o mapas interactivos de navegación.

En el siguiente ejemplo cada área de la misma imagen tiene un enlace que dirige a una página distinta.

Esto se logra implementando la etiqueta <map>, dentro de la cuál se definen áreas distintas por medio de la etiqueta <area>. Al final el mapa definido se asigna a la imagen por medio del atributo usemap de la etiqueta <img/>.

Atributos explicados

name

Establece el nombre del mapa en la etiqueta <map> para posteriormente asignárselo a la imagen.

shape

Determina el tipo de forma del área. Sus posibles valores son: rect, circle y polly.

coords

Establece las coordenadas del área según el tipo de forma.

  • rect: Para rectángulos las coordenadas se escriben como: x1, y1, x2, y2. Dónde x1 y y1 es la coordenada de la esquina superior izquierda del rectángulo, mientras que x2 y y2 es la coordenada de la esquina inferior derecha.
  • circle: Para circulos las coordenadas se escriben como: x, y, radius. Dónde x y y es la coordenada del centro del círculo, y radius la distancia entre el centro y el perímetro.
  • polly: Para polígonos las coordenadas se escriben como pares (x,y) de forma sucesiva, colocando tanto puntos como sean necesarios para completar la forma deseada: x1, y1, x2, y2, xn, yn.

href

Se utiliza para asignar la ruta de enlace al área correspondiente.

target

Establece el método apertura del enlace, sus valores pueden ser:

Valor Descripción
_blank Abre el enlace en una nueva ventana o pestaña.
_self Abre el enlace en el mismo documento.
_parent Abre el enlace en la ventana contenedora del iframe.
_top Abre el enlace en pantalla completa.
framename Abre el enlace dentro del iframe indicado.

title

Se utiliza para asignar un mensaje emergente sobre el área al pasar el puntero del ratón por encima.

usemap

Se implementa sobre la etiqueta <img/> para asignarle el nombre del <map> correspondiente y aplicar ese mapa a la imagen.
