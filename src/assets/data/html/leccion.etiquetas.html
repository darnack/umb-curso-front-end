Una etiqueta es una marca que representa un elemento HTML. Cada elemento HTML tiene una etiqueta asociada. Podemos crear un elemento usando su etiqueta correspondiente.

La mayoría de los elementos en HTML tienen una etiqueta de apertura y una etiqueta de cierre. Entre ambas etiquetas el elemento puede contener texto u otro elemento HTML.

Este elemento tiene una etiqueta de apertura y una etiqueta de cierre para rodear su contenido.

  • La etiqueta de apertura es <h1>.
  • La etiqueta de cierre es </h1>.

Las etiquetas pueden contener otras etiquetas, lo que permite estructurar contenido para crear formas más complejas.

Existen muchas etiquetas HTML. Las más usadas comúnmente son:

Etiqueta Descripción
<html> Todo el contenido de la página debe estar entre estas etiquetas.
<head> Contiene el título de la página que puedes ver en la pestaña de tu navegador y además contiene metadatos.
<body> Contiene todos los elementos visibles de la página web. Toda la estructura de la página debe estar entre estas etiquetas.
<div> Contenedor de elementos. Permite personalizar la estructura de un documento html.
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> Estos elementos representan encabezados o títulos de mayor a menor importancia o nivel en la página web.
<p> Párrafo. Ideal para contener grandes cantidades de texto.
<span> Etiqueta de texto. Puede utilizarse para pequeñas notaciones de texto.
<a> Enlace a otra página web o a secciones internas de la página web.
<img> Inserta una imagen.
<audio> Inserta un audio.
<video> Inserta un video.
<b> Texto en negrilla.
<i> Texto en cursiva.
<u> Texto subrayado.
<blockquote> Texto en formato de cita.
<hr> Línea horizontal. Permite crear separación entre diferentes contenidos.
<br> Salto de línea.
<form> Formulario. Permite capturar datos.
<input> Campo de entrada que permite capturar información para porteriormente enviarla a un servidor.
<label> Título de un campo de entrada.
<button> Botón de acción para dar clic.
<select> Lista desplegable.
<option> Elemento de la lista desplegable.
<textarea> Campo de texto multilínea.
<ol> Lista de elementos ordenados.
<ul> Lista de elementos no ordenados.
<li> Elemento de una lista.
<table> Permite crear una tabla.
<tr> Fila de una tabla
<td> Columna de una tabla.
<header> Cabecera de la página.
<nav> Menú de navegación.
<aside> Panel lateral.
<footer> Pie de página.
