Una de las características que más ha influido en el éxito de la Web ha sido, la posibilidad de unir los distintos documentos repartidos por todo el mundo por medio de enlaces hipertexto.

Uso de los enlaces

Los enlaces pueden utilizarsen de tres maneras:

Enlaces externos

Aquellos que dirigen a sitios fuera de nuestro propio sitio, para ello es necesario colocar la url completa del sitio externo en el atributo href.

Enlaces internos

Aquellos enlaces que permiten la navegabilidad dentro de nuesto propio sitio, para ello se deben utilizar url relativas en el atributo href.

Marcador

Permite que al dar clic, la página se desplaze visualmente a un punto específico, permitiendo al usuario ubicar rápidamente un contenido dentro una página. Para ello se debe colocar en el atributo href el símbolo # seguido del identificador del contenido que se quiere marcar. Por ejemplo, href="#contactar", para que funcione debe existir un elemento HTML visible cuyo atributo id correspondan al marcador, en el caso de este ejemplo: "contactar".


Uso especial de los enlaces

Mail To

Permite generar un enlace directo a la bandeja de correo electrónico para redactar un correo a una destinatario determinado. Para ello se debe colocar en el atributo href la palabra mailto seguido de dos puntos y el correo del destinatario.

JavaScript

Permite ejecutar funciones escritas en JavaScript. Para ello se debe colocar en el atributo href la palabra javascript seguido de dos puntos y el nombre de la función JavaScript.
