HTML es un lenguaje de marcado, permite definir la estructura de una página o aplicación web, el contenido que se va a mostrar en el navegador. Sus siglas significan "HyperText Markup Language", o Lenguaje de Marcado de Hipertexto.

Un archivo HTML está conformado por elementos HTML. Estos elementos son componentes individuales que nos permiten crear la estructura de la página.

Aquí tenemos un ejemplo de una página web muy sencilla:

Doctype

La declaración <!DOCTYPE html> es un document type declaration o DTD. Informa al navegador qué versión de HTML se usó para escribir el documento. En este caso el doctype corresponde a un documento HTML 5.

Html

La etiqueta <html> sirve para contener la cabecera y el cuerpo del documento HTML. Puede contener atributos como el idioma del documento.

Head

La cabecera, comprendida entre las etiquetas <head> y </head> contiene información del documento que no se ve en la pantalla del navegador. Principalmente la información encontrada en la cabecera es el título del documento, metadatos de la página, referencia a hojas de estilo y referencia a scripts.

Body

El cuerpo, comprendido entre las etiquetas <body> y </body> contiene todo lo que queremos que aparezca en la pantalla como texto, botones, controles, formularios, imágenes, vídeo y animaciones.

Meta

La etiqueta <meta> sirve para aportar información sobre el documento, como por ejemplo, el conjunto de caracteres soportados por el documento mediante el atributo charset.
