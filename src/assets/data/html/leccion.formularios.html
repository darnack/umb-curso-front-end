Los formularios permiten capturar información de una página y enviarla al servidor para su almacenamiento o procesamiento.

Se componen de campos de entrada y botones de acción que permiten la interacción con el usuario.

Etiqueta principal

Los atributos más importantes de la etiqueta <form> son:

Atributo Descripción
action Establece la url a la que se enviará la información, también permite ejecutar funciones JavaScript.
method Establece el método o verbo http con el cual se enviará la información: GET, POST, PUT, DELETE, etc.
enctype Cuando el valor del atributo method es post, este atributo es el tipo MIME del contenido que es usado para enviar el formulario al servidor.

Campos de entrada

Los campos de entrada o elementos <input/> permiten capturar datos del usuario, para que posteriormente sean enviados por medio del fomulario al servidor.

Existen varios tipos de campos de entrada:

Tipo Descripción
text Permiten la captura de texto.
password Permiten la captura de texto oculto.
number Límita la captura para que solo se ingresen números
range Permite seleccionar un valor dentro de un rango
radio Permite seleccionar una opción única.
checkbox Permite seleccionar una opción múltiple.
file Permite seleccionar un archivo del sistema.
color Permite seleccionar un color.
date Permite seleccionar una fecha.
month Permite seleccionar un mes.
week Permite seleccionar una semana.
time Permite seleccionar una hora específica.
datetime-local Permite seleccionar una fecha y hora.
hidden Campo oculto que permite guardar información temporalmente.

Select

Además de los inputs, existe el elemento <select> que permite la selección de una opción única dentro de un listado desplegable.

Botones de acción

Los botones de acción también son inputs, pero en lugar de comportarsen como campos de entrada, permiten la ejecución de una acción.

Tipo Descripción
submit Al hacer click serializa los datos capturados por los inputs y los envía al destino remoto configurado en la propiedad action del formulario.
button Al hacer click ejecuta una acción sin enviar los datos del formulario.
