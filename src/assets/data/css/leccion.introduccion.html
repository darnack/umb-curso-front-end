CSS (Cascading Style Sheets) es un lenguaje basado en reglas. Estas reglas nos permiten especificar las caraterísticas gráficas de los elementos HTML en la página web, a partir de selectores, propiedades y sus valores.

Estilos en línea

Se denomina estilo en línea cuando se asignan las propiedades gráficas directamente al elemento HTML. Para esto se utiliza la propiedad style, la cual es soportada por cualquier elemento HTML.

Etiqueta <style>

La etiqueta <style> permite contener estilos CSS que se puede aplicar a más de un elmento HTML a través de reglas de selección.

La etiqueta <style> debe incluirse dentro de la cabecera del documento HTML.

Archivos CSS

La manera más apropiada de implementar estilos CSS es por medio de un archivo CSS enlazado al documento HTML. Para ello, debemos incluir un elemento <link> en <head> y asignar la ubicación del archivo como el valor de href.

Por otro lado, el archivo de estilos debe tener extensión .css. Este contiene las reglas CSS que afectarán al documento HTML.
