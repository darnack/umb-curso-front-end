Las propiedades de texto permiten establecer el aspecto gráfico de la fuente, su color, estilo, espaciado, tamaño, etc. Algunas de las proiedades de texto más útiles son las siguientes:

Propiedad Descripción
font-family Define la fuente o tipografía del texto. Se puede especificar una lista de fuentes en orden de preferencia, en caso de que la primera opción no esté disponible.
font-size Define el tamaño de la fuente. Se puede especificar en píxeles, ems, porcentajes u otras unidades.
font-weight Define el grosor de la fuente. Se puede especificar valores como "normal" o "bold", o se pueden utilizar valores numéricos para un control más preciso.
font-style Define el estilo de la fuente. Se puede especificar valores como "normal", "italic" o "oblique".
text-align Determina la alineación del texto. Se puede especificar "left", "right", "center" o "justify".
text-decoration Agrega líneas decorativas al texto. Se pueden especificar valores como "none", "underline" o "line-through".
line-height Define el espaciado entre líneas de texto. Se puede especificar en píxeles, ems, porcentajes u otras unidades.
text-transform Permite transformar el texto a mayúsculas, minúsculas o letra capital. Se pueden utilizar los valores "uppercase", "lowercase" o "capitalize".
text-indent Establece la sangría de la primera leta de un párrafo. Se puede especificar en píxeles, ems, porcentajes u otras unidades.
letter-spacing Determina la separación entre caracteres. Se puede especificar en píxeles, ems, porcentajes u otras unidades.
word-spacing Determina la separación entre palabras. Se puede especificar en píxeles, ems, porcentajes u otras unidades.
text-shadow Crea un efecto de sobra detrás del texto. Se pueden especificar valores para la posición de la sombra, el desenfoque y el color.
color Define el color del texto. Se pueden especificar valores en hexadecimal o por nombre del color.

Fuentes seguras

Las fuentes seguras son aquellas que vienen incluidas en los navegadores de forma nativa. Se denominan seguras porque no dependen de una fuente externa, por lo tanto siempre estarán disponibles pase lo que pase. A continuación se muestra el listado completo de fuentes seguras.

  • Arial, sans-serif
  • Helvetica, sans-serif
  • Verdana, sans-serif
  • Tahoma, sans-serif
  • Trebuchet MS, sans-serif
  • Times New Roman, serif
  • Georgia, serif
  • Garamond, serif
  • Courier New, monospace
  • Lucida Console, monospace

Fuentes externas

Si deseamos implementar una fuente especial, que no esté incluida en el navegador, debemos incluirla desde un origen externo.

Para ello se debe hacer uso de la etiqueta <link> y por medio del atributo href referenciar el origen de la fuente.

Fuentes locales

Otra manera de implementar fuentes es referenciándolas localmente a partir de sus archivos de origen.

Para ello se debe hacer uso de la etiqueta @font-face como una regla más de CSS, asignando un nombre y origen a la fuente.
