La propiedad display:flex es ampliamente utilizada para estructurar contenido HTML. Esta permite darle comportamineto en bloque al contenedor, pero al mismo tiempo le asigna comportamiento en línea al contenido, y además, mantiene dinámicamente las proporciones del contenido sin importar la cantidad de elementos contenidos.

Flex direction

La propiedad flex-direction controla el main axis, es decir, la dirección en la cual los elementos son dibujados dentro de un contenedor flex.

Flex wrap

El comportamiento por defecto de flex hace que los elementos se ajusten al ancho total del contenedor, esto causa que entre más elementos tenga el contenedor, el espacio disponible para cada elemento sea menor, lo que deforma a los elementos alargándolos.
