Los métodos de cadena permiten realizar operaciones muy útiles sobre cadenas de texto. Estos métodos se utilizan como métodos de extensión, es decir, se invocan como métodos de la misma cadena: cadena_de_texto.metodo();.

Método Descripción
concat() Recibe una o varias cadenas de texto que une para devolver una sola.
startsWith() Determina si el comienzo de una cadena coincide con el valor de entrada del método.
endsWith() Determina si el final de una cadena coincide con el valor de entrada del método.
includes() Determina si la cadena contiene al comienzo, al final, o de forma intermedia, el valor de entrada del método.
indexOf() Si el texto de entrada existe dentro de la cadena original, devuelve el índice donde comienza el texto encontrado, de lo contrario devuelve -1.
lastIndexOf() Si el texto de entrada existe dentro de la cadena original, devuelve el índice donde termina el texto encontrado, de lo contrario devuelve -1.
padStart() Rellena la cadena de texto a la izquiera tantas posiciones como se indique, y con el caracter que se indique.
padEnd() Rellena la cadena de texto a la derecha tantas posiciones como se indique, y con el caracter que se indique.
repeat() Devuelve la misma cadena tantas veces como se indique.
split() Divide la cadena por medio de un carcater separador.
substring() Devuelve un segmento de la cadena con base a los índices de comienzo y fin indicados.
toLowerCase() Convierte una cadena a minúscula.
toUpperCase() Convierte una cadena a mayúscula.
toString() Convierte un valor que no es cadena string a su representación tipo cadena.
trim() Elimina los espacios al comienzo y fin de una cadena de texto.
