JavaScript cuenta con un amplio conjunto de operadores que nos pueden resultar de mucha utilidad.

Operadores aritméticos

Nombre Ejemplo Descripción
Suma a + b Suma el valor de a al valor de b.
Resta a - b Resta el valor de a al valor de b.
Multiplicación a * b Multiplica el valor de a al valor de b.
División a / b Divide el valor de a entre el valor de b.
Módulo a % b Devuelve el residuo de la división de a entre b.
Exponenciación a ** b Eleva a a la potencia de b.

Operadores de asignación

Nombre Ejemplo Descripción
Asignación x = y Asigna un valor a una variable u objeto.
Adición x += y x = x + y
Resta x -= y x = x - y
Multiplicación x *= y x = x * y
División x /= y x = x / y
Residuo x %= y Se asigna a x el residuo de la división entre x y y.
Exponenciación x **= y Se asigna a x el resultado de x elevado a la y.

Operadores de comparación

Nombre Ejemplo Descripción
Igualdad a == b Comprueba si el valor de a es igual al de b. No comprueba tipo de dato.
Identidad a === b Comprueba si el valor y tipo de dato de a son iguales al de b.
Desigualdad a != b Verdadero si el valor de a es diferente al de b. No comprueba tipo de dato.
No Idéntico a !== b Verdadero si el valor y tipo de dato de a son diferentes al de b.
Mayor que a > b Comprueba si el valor de a es mayor que el de b.
Mayor igual que a >= b Comprueba si el valor de a es mayor o igual que el de b.
Menor que a < b Comprueba si el valor de a es menor que el de b.
Menor igual que a <= b Comprueba si el valor de a es menor o igual que el de b.

Operadores unarios

Nombre Ejemplo Descripción
Incremento a++ Usa el valor de a y luego lo incrementa. También llamado postincremento.
Decremento a-- Usa el valor de a y luego lo decrementa. También llamado postdecremento.
Incremento previo ++a Incrementa el valor de a y luego lo usa. También llamado preincremento.
Decremento previo --a Decrementa el valor de a y luego lo usa. También llamado predecremento.
Resta unaria -a Cambia de signo a la variable a.

Operadores binario

Nombre Ejemplo Descripción
AND a & b Devuelve verdadero si ambos operandos son verdaderos.
OR a | b Devuelve verdadero si al menos un operando es verdadero.
XOR a ^ b Devuelve verdadero si ambos operandos son diferentes.
NOT ~a Invierte los bits del operando (por ejemplo, 000101 pasa a 111010). Trunca a 32 bits.
LEFT SHIFT a << b Desplaza el primer operando el número especificado de bits a la izquierda. Ejemplo: 9<<2 produce 36, porque 1001 desplazado 2 bits a la izquierda se convierte en 100100, que es 36.
RIGHT SHIFT a >> b Desplaza el primer operando el número especificado de bits a la derecha. Ejemplo: 9>>2 produce 2, porque 1001 desplazado 2 bits a la derecha se convierte en 10, que es 2.
RIGHT SHIFT sin signo a >>> b Similar al desplazamiento a la derecha RIGHT SHIFT, excepto que todos los bits vacíos a la izquierda se rellenan con 0s. Esto significa que el resultado de este operador siempre es un entero positivo y siempre trata el entero que se desplaza como un entero sin signo.
