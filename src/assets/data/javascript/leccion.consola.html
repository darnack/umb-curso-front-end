La consola de depuración es una herramienta de desarrollo de software que permite evaluar el correcto funcionamiento de nuestras aplicaciones web.

Para acceder a la consola del navegador debemos presionar la tecla F12 y en el panel desplegado seleccionar la pestaña console. Allí se podrá visualizar los mensajes de consola escritos desde el aplicativo.

JavaScript cuenta con el componente console que permite escribir mensajes en la consola de depuración. Gracias a esto podemos evaluar y verificar el valor de variables y objetos en tiempo de ejecución, medir tiempos de respuesta y verificar en general el correcto comportamiento de la aplicación o sitio web.

Mensajería

Estos métodos permiten escribir mensajes en la consola de depuración, resulta muy útil para visualizar el valor de un objeto o variable en tiempo de ejecución, o para dejar un mensaje de seguimiento que nos permita reconocer que la ejecución de código a llegado a un punto determinado.

console.log()

Permite escribir un mensaje en consola, puede recibir un número indeterminado de parámetros de entrada, los cuales pueden ser un mensaje de texto, una variable o un objeto.

console.info()

Igual a console.log(), permite escribir un mensaje de nivel information.

console.warn()

Permite escribir un mensaje de nivel warning que se visualiza de color amarillo.

console.error()

Permite escribir un mensaje de nivel error que se visualiza de color rojo.

console.clear()

Elimina todos los mensajes de la consola de depuración.

Estructurados

Estos métodos permiten escribir datos de forma estructurada sobre la consola de depuración, lo cuál permite un mejor análisis de la información que maneja la aplicación.

console.table()

Permite visualizar un objeto de forma estructurada en formato tabla.

console.trace()

Permite visualizar el stacktrace o pila de llamados.

Conteo

Estos métodos permiten medir de forma concreta la cantidad de veces que se repite la ejecución de un bloque de código determinado.

console.count()

Permite llevar un conteo por medio de una etiqueta. De esta manera podemos identificar cuantas veces se ha ejecutado cada etiqueta de manera independiente.

console.countReset()

Reinicia el conteo de una etiqueta.

Tiempo

Estos métodos permiten medir de forma concreta el tiempo que toma la ejecución de un bloque de código determinado.

console.time()

Inicia un temporizador, puede recibir un nombre como parámetro, lo que permite manejar hasta 10.000 temporizadores de manera simultánea.

console.timeLog()

Toma el tiempo transcurido del temporizador en milisegundos.

console.timeEnd()

Detiene el temporizador y devuelve el tiempo transcurrido en milisegundos.
